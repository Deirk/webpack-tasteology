# Tasteology Webpage - HTML, TS, Tailwind and Webpack 

# Build
Steps to build and start app

1. Run command ``` npm install ```
2. Run command ``` npm run build ```

# Dev
Steps to start dev app

1. Run command ``` npm install ```
2. Run command ``` npm run dev ```
